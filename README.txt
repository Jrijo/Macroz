App: Macroz

Description: This calorie counter is a way to count your daily caloric intake using an easy to use caloric counter. Calorie counting is an easy way for you to manage your weight. By entering the macronutrient goals and then manipulating them whenever you grab some grub you can easily keep track of your health and wellness goals.
